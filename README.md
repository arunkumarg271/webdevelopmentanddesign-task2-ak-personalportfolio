# webdevelopmentanddesign-task2-ak-personalportfolio
This repository contains my personal portfolio website created using basic web technologies. The purpose of this project is to present my academic background, technical skills, projects, and certifications in a single webpage.  The website is designed to be simple, clear, and easy to navigate.

**Project Overview**

The portfolio includes different sections such as profile details, areas of interest, education, skills, projects, and contact information.
It also contains hover effects and smooth navigation to improve user experience.
This project helped me understand how HTML and CSS work together to create a structured and visually appealing webpage.

**Technologies Used**

-> HTML5 for creating the webpage structure

-> CSS3 for styling, layout, and hover effects

-> Profile section with image

**Education and skills details**
